# 🌀 Naruto Daily Tracker

A fun, interactive daily task tracker inspired by Naruto, built with Node.js, Express, and SQLite. Perfect for kids to track their daily missions!

## 🌟 Features

- **Ninja Missions**: Pre-loaded tasks with emojis (Wake up, Brush teeth, etc.)
- **Database Storage**: All entries saved in SQLite database
- **Naruto Theme**: Blue background, orange accents, motivational messages
- **CRUD Operations**: Add, edit, delete tasks
- **Reports**: Export to CSV or print to PDF
- **Image Upload**: Add custom images or use default Naruto face

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/naruto-daily-tracker.git
   cd naruto-daily-tracker
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser to `http://localhost:3000`

## 🎮 How to Use

1. **Add Missions**: Select a task from the dropdown and add your entry
2. **Track Progress**: View all entries in the mission scroll
3. **Edit/Delete**: Use the action buttons to modify entries
4. **Export**: Download CSV reports or print PDFs
5. **Upload Images**: Add custom images or enjoy the default Naruto face

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: SQLite3
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with Naruto theme
